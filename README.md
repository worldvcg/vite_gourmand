# Vite & Gourmand

Projet ECF — application web vitrine + commande de menus.

## Lancement (macOS + MAMP)
1. Copier le dossier dans `/Applications/MAMP/htdocs/vite_gourmand`
2. Démarrer MAMP (Apache + MySQL)
3. Accéder à `http://localhost:8888/vite_gourmand/front/index.html`
4. Base de données : créer `vite_gourmand` dans phpMyAdmin
